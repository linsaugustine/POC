Neuron ESB
PubSub Object Sample

Overview

The Pub-Sub Object sample demonstrates publish-subscribe messaging with an object payload

In publish-subscribe, messages from a sender can be received by any number of parties who have a matching subscriptions. 

This sample is located in the samples hierarchy at Samples\Features\MessagingPatterns\PubSub\PubSubObject.

Setup Instructions

Before running this sample, follow these instructions:

1.       Open the sample solution in Visual Studio and add a reference to ESB.DLL.

2.       Build the solution. 

Run Instructions

1. Start the ESB Service running with sample configuration SampleContacts.esb

2. Open the solution file in Visual Studio. Start up all projects.

3. Once all programs have connected to the ESB, press Enter on the sender to initiate communication.

4. As the sample runs, the sender's messages are received by multiple receivers with matching subscriptions.
